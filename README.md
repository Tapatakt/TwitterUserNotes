# Twitter User Notes

Плагин для Chrome/Brave, который позволяет оставлять личные комментарии (заметки) к пользователям Twitter / X.

> ⚠️ **Важно:** плагин навайбкожен.

## Что умеет

- При наведении на ссылку профиля (`x.com/Username` или `twitter.com/Username`) через 0.5 секунд справа экрана появляется текстовое поле с твоим комментарием к этому пользователю.
- Если комментарий раньше не сохранялся — поле пустое.
- **Enter** или потеря фокуса — сохраняет комментарий.
- **Ctrl** (пока тултип открыт) — закрепляет его. Закреплённый тултип не исчезает при уходе мыши и закрывается только:
  - по клику снаружи,
  - по Enter,
  - при наведении на другую ссылку профиля.
- Пустой комментарий при сохранении удаляет запись из хранилища.

## Установка (ручная)

1. Открой `chrome://extensions/` (или `brave://extensions/`).
2. Включи **Режим разработчика** (переключатель справа сверху).
3. Нажми **Загрузить распакованное расширение** (Load unpacked).
4. Выбери папку с этим проектом (где лежит `manifest.json`).

## Структура

```
├── manifest.json                 — манифест расширения
├── README.md                     — этот файл
├── .gitignore                    — исключения для Git
└── content-scripts/
    ├── storage.js                — работа с chrome.storage.local
    ├── tooltip.js                — UI тултипа (справа экрана)
    ├── link-detector.js          — обнаружение ссылок на профили
    └── content.js                — точка входа, связка модулей
```

## Технические детали

- **Manifest V3**
- Content script работает только на `x.com` и `twitter.com`
- Хранение — `chrome.storage.local` (без SQL, без синхронизации)
- Исключены системные пути: `/home`, `/explore`, `/notifications`, `/tos`, `/privacy`
- Обработка событий в фазе захвата (`capture: true`), чтобы встроенные скрипты Twitter не перехватывали мышь раньше нас
